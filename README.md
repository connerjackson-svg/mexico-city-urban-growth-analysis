# Mexico City Urban Growth Analysis

Geospatial machine learning project analyzing urban expansion in Mexico City and identifying priority zones for green infrastructure investment.

## Tools
Google Earth Engine, Landsat, Random Forest, GIS, Spatial Modeling

## Highlights
- Detected 249.91 km² of new urban land (2005–2022)
- Predicted future development risk zones
- Identified top priority areas for green infrastructure
